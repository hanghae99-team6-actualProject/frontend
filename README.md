<h1 align=left><strong>밍기적</strong>을 소개합니다.</h1>

오랜 시간 좌식 생활을 하시거나 운동을 시작하지만 작심삼일로 끝나곤 하나요?

건강을 지키기 위해 꼭 거창한 운동을 할 필요는 없어요. 일상 속에서 여러분이 할 수 있는 운동을 꾸준히 한다면 하루하루 달라진 여러분을 발견할 수 있을 거예요.

<h2>1. 운동을 기록하고 루틴을 구성할 수 있습니다.</h2>
<br />
<div>
<p align=center>🏃 액션과 루틴개념을 기반으로 운동을 기록할 수 있습니다.</p>
<p align=center>🏃 액션 : 실제 사용자가 실시하는 단일 운동 / 루틴 : 액션의 조합</p>
<p align=center>🏃 프리셋으로 제공되는 기본 루틴 외, 직접 루틴을 구성할 수 있습니다.</p></div>

<h2>2. 운동 모임을 개설하고 모임 참여자들과 채팅으로 소통할 수 있습니다.</h2>
<div>
<p align=center>📑 게시판 : 게시판을 통해 나와 비슷한 사람들과 소통할 수 있습니다. </p>
<p align=center>😮 필터 : 필터 기능을 통해 내 지역, 지금 인기있는 모임 등을 바로 볼 수 있습니다. </p>
<p align=center>🗺 지도 : 카카오 지도 API를 기반으로 상세한 모임 위치를 적용할 수 있습니다.</p>
<p align=center>💬 채팅 : 모임 참여자들과 실시간 채팅으로 소통할 수 있습니다.</p>
</div>

<div align=center>
  <h2>4. 운동 이력을 조회할 수 있습니다.</h2>
  <div>
  <p align=center><strong>Habit Tracker History</strong> : 운동 습관 형성을 돕는 Habit Tracker는 한 달을 기준으로 일자 템플릿이 표시됩니다.
        루틴을 완수하면 템플릿 내 완수 일자에 마크가 적용됩니다.</p>
   <p align=center><strong>Graph History</strong> : 운동 이력을 막대그래프로 시각화해서 쉽게 확인할 수 있는 Graph History 주 단위로 표시됩니다. 완수한 액션 수 또는 루틴 수를 기준으로 각각 확인할 수 있습니다.</p>
</div>
